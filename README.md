# question_classifier
This module classifies different What, When, Who and Affirmative (Yes-No) questions. 

# INSTALLATION
```
pip install -r requirements.txt
```
# Classify Questions
```
python classify.py

Type exit() to exit
Enter question:
Who are you ?
who
Enter question:
What time doest the train arrive?
when
Enter question:
Why did Kattapa kill Bahubali?
unknown
Enter question:
Do you love me?
affirmation
Enter question:
Narendra Modi is prime minister of?
who
```
